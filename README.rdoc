= model_notifier

Sends an email when a model gets created.

= Configuration 

Puts this inside an initializer

  config/initializers/model_notifier.rb

  ModelNotifier.configure do
    model :foo
    recipients "corey@example.com"
    subject "test email from corey"
  end
